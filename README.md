# IPA Keyboard

## Overview

The IPA Keyboard presents all symbols in a randomised layout, encouraging users to rely on their phonetic knowledge rather than recognition or memorised positions.  
It is intentionally simple, uncluttered, and exam‑friendly.

## Key Features

- **Randomised IPA symbols**  
  The keyboard presents the IPA symbols in randomised orders instead of presenting it according to the IPA chart.

- **No labels or categories**  
  Symbols appear without three-term labels, allowing learning without prompts.

- **Exam‑ready design**  
  Built specifically with digital phonetics exams in mind.

## Who Is This For?

- **Lecturers**  
  Useful for creating online phonetics assessments requiring unbiased symbol input.

- **Students**  
  Ideal for consolidating IPA knowledge and practising IPA transcription before exams.


## How to Use

1. Click any IPA symbol to insert it into the canvas.  
2. Use **CLEAR** to reset your input.  
3. Copy your transcription for submission, practice, or feedback.
